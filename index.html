The PeopleSoft Adminstrator Wiki

Why build a wiki

  1. Share technical "How To" articles
  2. Build a community repo of documentation
  3. Easily share short tips/tricks with a large audience
  4. Learn Git

How to Contribute

Clone and Branch

  1. git clone https://github.com/psadmin-io/psadmin-wiki
  2. git checkout -b my-changes

Add Your Article

Add a page

  1. touch posts/pca/hiera_hashes.md
  2. Add the front-matter
  3. Write your post
  4. Add your article link to _data\pca.yml

Configure Article

front-matter

---
title: Sample psft_customizations.yaml
layout: en
permalink: /posts/pca/sample_yaml/
---

_data/pca.yml

Sample psft_customizations.yaml:  "/posts/pca/sample_yaml/"
DPK Cleanup Process:              "/posts/pca/cleanup/"

Test Your Changes

Run Jekyll Locally

Test your new article by browsing the wiki locally.

  1. gem install jekyll
  2. bundle exec jekyll serve
  3. Open http://localhost:4000

Publishing Your Changes

Push and Pull

  1. git add .
  2. git commit -m "My Changes"
  3. git push origin my-changes
  4. Create a Pull Request
  5. Verify your branch builds correctly
